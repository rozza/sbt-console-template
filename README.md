# sbt-console-template

Get started with Scala and use Intellij IDEA (tested on 11)

* Scala 2.10.2
* Specs 2.1.1

## Using

Run the app

    ./sbt run

Run tests

    ./sbt test


## Convert to IDEA project

1. Start sbt, generate IDEA project

    ./sbt gen-idea

2. Open the project in IDEAj from the menu, File -> Open Project ...
